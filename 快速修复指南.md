# 快速修复：将 Python 默认版本改为 3.10

## 问题
在 CMD 中运行 `python --version` 显示 Python 3.14.2，需要改为 3.10。

## 解决方案

### 方案一：使用批处理脚本（推荐，最简单）

1. **以管理员身份运行** `fix_python_path.bat`
   - 右键点击 `fix_python_path.bat`
   - 选择"以管理员身份运行"
   - 按照提示操作

2. **关闭所有终端窗口并重新打开**

3. **验证**：
   ```cmd
   python --version
   ```
   应该显示 `Python 3.10.11`

---

### 方案二：手动编辑系统环境变量

1. **打开环境变量设置**：
   - 按 `Win + X` → 选择"系统"
   - 或：右键"此电脑" → "属性"
   - 点击"高级系统设置"
   - 点击"环境变量"

2. **编辑系统 PATH**：
   - 在"系统变量"区域找到 `Path`
   - 选中后点击"编辑"
   - 查找并**删除**以下两项：
     - `C:\Python314\`
     - `C:\Python314\Scripts\`
   - 点击"确定"保存

3. **关闭所有终端窗口并重新打开**

4. **验证**：
   ```cmd
   python --version
   where python
   ```

---

### 方案三：使用 PowerShell 脚本（需要管理员权限）

1. **以管理员身份打开 PowerShell**

2. **运行脚本**：
   ```powershell
   cd C:\Users\hlin2\jewelry-ai-erp
   .\set_python310_default.ps1
   ```

3. **关闭所有终端窗口并重新打开**

---

### 方案四：临时使用 Python 3.10（不修改环境变量）

如果不想修改系统设置，可以使用：

1. **使用 `py` 命令**（已配置为默认 3.10）：
   ```cmd
   py --version
   py your_script.py
   ```

2. **使用完整路径**：
   ```cmd
   C:\Users\hlin2\AppData\Local\Programs\Python\Python310\python.exe --version
   ```

3. **创建别名**（在项目中使用 `python310.bat`）

---

## 验证修复是否成功

运行以下命令：

```cmd
python --version        # 应该显示 Python 3.10.11
where python            # 第一个路径应该是 Python 3.10
py --version            # 应该显示 Python 3.10.11
```

- ✅ `py` 命令：→ Python 3.10.11（已配置）
