# 珠宝ERP标签打印功能 - 实施文档

## 一、需求概述

在ERP系统中增加珠宝吊牌标签打印功能，支持从商品详情页、入库单、库存列表等位置一键打印标签。

### 标签规格

- **尺寸**：30mm × 40mm 蝴蝶型珠宝吊牌
- **打印机**：GODEX 科诚碳带打印机
- **连接方式**：USB 连接到店铺电脑

### 标签内容（参考现有格式）

```
┌──────────────────────┐
│  |||||||||||||||||||  │  ← 条形码
│  F00147550           │  ← 商品编号
│  主石:0.01/1p        │
│  副石:0.13/43p       │
│                      │
│  足金5D钻石挂坠       │  ← 商品名称
│  工费: 35.00/克      │  ← 克重工费
│  其他工费: 830.00/件  │  ← 件工费
│  金重: 2.59g         │  ← 金重（加粗加大）
└──────────────────────┘
```

## 二、技术方案

### 选用方案：C-Lodop 打印控件

C-Lodop 是国内企业 Web 应用最成熟的打印方案，支持所有主流热敏/碳带打印机。

**工作原理：**

```
浏览器(ERP页面)
    ↓ JavaScript 调用
C-Lodop 本地服务 (localhost:8000/18000)
    ↓ 系统打印接口
GODEX 打印机驱动
    ↓
打印机出标签
```

**优势：**
- 支持毫米级精确定位，适配珠宝吊牌特殊尺寸
- 支持条形码、二维码生成
- 支持所有主流浏览器（Chrome/Edge/Firefox）
- 免费使用（商用版有水印提示，可购买注册码去除，约500元）
- 国内文档丰富，社区活跃

## 三、安装步骤（运维/同事执行）

### 步骤 1：安装 GODEX 打印机驱动

> 如果打印机已经能正常打印（比如用 GoLabel 软件），说明驱动已安装，跳过此步。

1. 访问 GODEX 官网下载驱动：https://www.godex.com.cn/downloads
2. 选择对应型号，下载 Windows 驱动
3. 安装后确认在 Windows「设置 → 打印机和扫描仪」中能看到 GODEX 打印机

### 步骤 2：安装 C-Lodop 打印服务

1. 访问官网下载：http://www.lodop.net/download.html
2. 下载「C-Lodop 云打印服务」（Windows 版，约 5MB）
3. 双击安装，一路默认下一步即可
4. 安装完成后，C-Lodop 会作为 Windows 服务自动启动
5. 验证安装成功：
   - 打开浏览器访问 http://localhost:8000
   - 如果看到 C-Lodop 服务页面，说明安装成功

### 步骤 3：验证打印机连接

1. 打开浏览器访问 http://localhost:8000
2. 点击页面上的「打印测试页」
3. 在弹出的打印机选择中选择 GODEX 打印机
4. 确认能正常出纸

## 四、前端开发任务

### 4.1 引入 Lodop 脚本

在前端项目中添加 Lodop 工具文件：

```javascript
// frontend/src/utils/lodopPrint.js

/**
 * 加载 Lodop 打印控件
 * C-Lodop 服务运行在本地 localhost:8000/18000
 */
export function getLodop() {
  // 优先使用 C-Lodop (云打印)
  try {
    const LODOP = window.getCLodop()
    if (LODOP) return LODOP
  } catch (e) {
    console.warn('C-Lodop not available:', e)
  }
  return null
}

/**
 * 打印珠宝吊牌标签
 * @param {Object} product - 商品数据
 * @param {string} product.barcode - 条形码编号（如 F00147550）
 * @param {string} product.productName - 商品名称（如 足金5D钻石挂坠）
 * @param {number} product.goldWeight - 金重（克）
 * @param {number} product.laborCost - 工费（元/克）
 * @param {number} product.pieceLaborCost - 件工费（元/件）
 * @param {string} product.mainStone - 主石信息（可选）
 * @param {string} product.sideStone - 副石信息（可选）
 */
export function printJewelryLabel(product) {
  const LODOP = getLodop()
  if (!LODOP) {
    alert('打印控件未安装，请先安装 C-Lodop 打印服务')
    window.open('http://www.lodop.net/download.html')
    return
  }

  // 初始化打印任务
  LODOP.PRINT_INIT('珠宝标签')

  // 设置标签纸尺寸（单位 mm）
  LODOP.SET_PRINT_PAGESIZE(1, '30mm', '40mm', '')

  // 打印条形码
  LODOP.ADD_PRINT_BARCODE('2mm', '2mm', '26mm', '8mm', 'Code128Auto', product.barcode)

  // 打印商品编号
  LODOP.ADD_PRINT_TEXT('11mm', '2mm', '26mm', '3mm', product.barcode)
  LODOP.SET_PRINT_STYLEA(0, 'FontSize', 7)

  // 主石/副石信息（如果有）
  let yPos = 15
  if (product.mainStone) {
    LODOP.ADD_PRINT_TEXT(`${yPos}mm`, '2mm', '26mm', '3mm', `主石:${product.mainStone}`)
    LODOP.SET_PRINT_STYLEA(0, 'FontSize', 6)
    yPos += 3
  }
  if (product.sideStone) {
    LODOP.ADD_PRINT_TEXT(`${yPos}mm`, '2mm', '26mm', '3mm', `副石:${product.sideStone}`)
    LODOP.SET_PRINT_STYLEA(0, 'FontSize', 6)
    yPos += 3
  }

  // 商品名称（加粗）
  LODOP.ADD_PRINT_TEXT(`${yPos + 1}mm`, '2mm', '26mm', '4mm', product.productName)
  LODOP.SET_PRINT_STYLEA(0, 'FontSize', 8)
  LODOP.SET_PRINT_STYLEA(0, 'Bold', 1)

  // 工费信息
  LODOP.ADD_PRINT_TEXT(`${yPos + 6}mm`, '2mm', '26mm', '3mm',
    `工费: ${product.laborCost.toFixed(2)}/克`)
  LODOP.SET_PRINT_STYLEA(0, 'FontSize', 7)

  if (product.pieceLaborCost > 0) {
    LODOP.ADD_PRINT_TEXT(`${yPos + 9}mm`, '2mm', '26mm', '3mm',
      `其他工费: ${product.pieceLaborCost.toFixed(2)}/件`)
    LODOP.SET_PRINT_STYLEA(0, 'FontSize', 7)
  }

  // 金重（加粗加大，底部显示）
  LODOP.ADD_PRINT_TEXT(`${yPos + 13}mm`, '2mm', '26mm', '5mm',
    `金重: ${product.goldWeight.toFixed(product.goldWeight < 10 ? 2 : 3)}g`)
  LODOP.SET_PRINT_STYLEA(0, 'FontSize', 10)
  LODOP.SET_PRINT_STYLEA(0, 'Bold', 1)

  // 选择打印机并打印
  // LODOP.SET_PRINTER_INDEX('GODEX') // 可指定打印机名称
  LODOP.PREVIEW() // 开发时用预览，正式上线改为 LODOP.PRINT()
}
```

### 4.2 在 HTML 中引入 C-Lodop 脚本

在 `frontend/index.html` 的 `<head>` 中添加：

```html
<!-- C-Lodop 云打印服务 -->
<script src="http://localhost:8000/CLodopfuncs.js"></script>
<script src="http://localhost:18000/CLodopfuncs.js"></script>
```

### 4.3 添加打印按钮

在库存详情、入库单等页面添加打印按钮：

```jsx
import { printJewelryLabel } from '../utils/lodopPrint'

// 在商品行或详情页中
<button onClick={() => printJewelryLabel({
  barcode: product.barcode || product.product_code,
  productName: product.product_name,
  goldWeight: product.weight,
  laborCost: product.labor_cost,
  pieceLaborCost: product.piece_labor_cost || 0,
  mainStone: product.main_stone || '',
  sideStone: product.side_stone || '',
})}>
  🏷️ 打印标签
</button>
```

## 五、可能需要的后端改动

### 5.1 商品编码/条码字段

确认数据库中商品表是否有 `barcode` 或 `product_code` 字段。如果没有，需要添加：

```sql
ALTER TABLE inventory ADD COLUMN barcode VARCHAR(20);
```

### 5.2 批量打印 API（可选）

如果需要批量打印（比如一次入库多个商品），可以添加一个返回批量商品数据的 API。
